This project is DrivEx, a web-based application.
Users can log their driving sessions, view statistics, and analyze their driving habits using charts and tables.

DrivEx includes the following features:
-User login and registration
- Add driving sessions (manual entry)
- Dashboard with statistics:
  - Total distance
  - Total trips
  - Average speed
  - Driving history
- Charts based on:
  - Weather
  - Road type
  - Traffic condition
  - Visibility
- Clear driving history option
- Responsive interface (desktop and mobile)

Database Configuration (config.php)
The file `config.php` contains the database connection settings, including the database password.
